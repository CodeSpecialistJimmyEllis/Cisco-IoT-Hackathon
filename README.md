HackathonDate: October 10-12, 2014  
  Cisco themselves came to 222 Merchandise Mart in Chicago IL the tech incubator called 1871( https://1871.com/ ) and threw a hackathon asking us to code there 
newest router technology to make a project. I have a project with a website created by my team mates and an api call to one of the routers 
that gave you location information. Sadly, I have only an API call. I don't even have information used and displayed in a great way doing something. 
I learned alot. But this is no spectacular project. But this is a C# api call to a router asking for key information about the people and computers around you.

---------------
(2026 Update)
Alright so I had a team this time instead of just myself coding something while teams worked as hard as they could. They made the front end, I was going to make the back end API call.

API calls back then for me were hard. Every other hackathon I'd have to call a new API and then I'd have to call it in a different language using different libraries for different reasons.
It really takes a while before you begin to see the similarities that are completely obvious to me now in all API calls. Back then call an API was an art. Now you can just get AI to write
the API call for you. I mean, not the whole app, if you just ask it to write a simple API call for you it will write good code because you had it write something obvious. And then if it's not 
what you want you at least DO HAVE a successful non buggy API call. And then you can change it around and make it work for you and your project.

BUT BACK THEN WE HAD NONE OF THAT.
The guys that made the front end successfully wrote a good front end.

My back end sucked. I hardly understood what the new CISCO technology actually did. I didn't get it. I'm A+ Certified but I Don't have a CCNA so I was just absolutely defeated. In the end
I wrote an API call that would display some data and I thought that was a pretty big deal. But now looking back it really didn't do anything innovative with that data. So well, it was just
a novice accomplishment.
